📖 Setup Guide

Step 1 — Google Sheet
Create a new Google Sheet. Give it any name (e.g. "ITMIS Register 2026").

Step 2 — Apps Script
Extensions → Apps Script → paste Code.gs content → Save.

Step 3 — Deploy
Deploy → New deployment → Web App → Execute as: Me → Access: Anyone → Deploy → Copy URL.

Step 4 — Set token
Change SECRET_TOKEN in Code.gs to something private, then paste the same value above.

Step 5 — GitHub Pages
Push index.html, manifest.json, sw.js, and icon files to a GitHub repo → Settings → Pages → Deploy from main branch.

Step 6 — Install PWA
Desktop: Open in Chrome → address bar → Install icon. Android: Open in Chrome → menu → "Add to Home Screen".
